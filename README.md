Setup requirements
=========

```
  pyenv install 3.12.0
  pyenv virtualenv 3.12.0 ansible-env
  pyenv activate ansible-env
  pip install --upgrade pip
  pip install -r requirements.txt
```

Role Name
=========

UFW basic setup. Default behavior is restrictive: deny all connections !

Role Variables
--------------

You must specify the new user(s) details (name, pwd, authorized ssh key).


 
Tests
----------------

```
  # create the instance
  molecule create

  # apply the playbook
  molecule converge

  # test the playbook
  molecule verify

  # destroy the instance
  molecule destroy  
```
